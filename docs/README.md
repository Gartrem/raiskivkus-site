# Райский вкус — Сайт доставки еды

Статический сайт для магазина "Райский вкус" (п. Октябрьский).
Суши, роллы, пицца и WOK блюда.

## Развёртывание на GitHub Pages

1. Создайте новый репозиторий на GitHub
2. Загрузите содержимое папки `docs/` в корень репозитория (или всю папку `docs/`)
3. Перейдите в **Settings -> Pages**
4. Source: **Deploy from a branch**
5. Branch: `main`, Folder: `/docs` (или `/` если загрузили содержимое в корень)
6. Нажмите **Save**
7. Через пару минут сайт будет доступен по адресу `https://ваш-логин.github.io/имя-репозитория/`

## Обновление данных

Для обновления меню, контактов или акций отредактируйте JSON файлы:

- `data/products.json` — список товаров (название, цена, описание, категория, изображения)
- `data/settings.json` — контакты, адрес, время работы, ссылки на VK и Telegram
- `data/promotions.json` — акции

## Структура

```
docs/
  index.html          — главная страница
  404.html            — SPA fallback для роутинга
  .nojekyll           — отключает Jekyll обработку
  data/               — статические данные (JSON)
  assets/             — стили, скрипты, изображения
  images/             — категории изображений
```
